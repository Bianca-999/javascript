¿Qué es HTML?

siempre dicen, "html es un lenguaje de marcado" pero nadie te explica qué es un lenguaje de marcado, así que aquí va: un lenguaje de marcado es un sistema de codificación que se utiliza para estructurar y presentar contenido en la web. ¿qué significa en palabras sencillas? que es una forma de darle estructura y formato a los contenidos de una página web. Está compuesto por ETIQUETAS ¿ok? guarda bien esa palabra en tu cabeza. Las etiquetas son como instrucciones que le dicen al navegador cómo mostrar el contenido. Por ejemplo, la etiqueta <p> se utiliza para crear un párrafo, la etiqueta <h1> se utiliza para crear un encabezado de nivel 1, y así sucesivamente hasta el <h6>. HTML es la base de cualquier sitio web.
